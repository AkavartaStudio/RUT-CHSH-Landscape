## Supplementary Figures

### Figure S1: Control Comparison of Random vs Optimized Parameter Configurations

**Figure S1**: Control comparison of random vs optimized parameter configurations.

**(A)** Histogram of |S| values from N = 100 random parameter configurations (gray) vs N = 20 replications of optimized configuration (red vertical line at |S| = 2.778). Random configurations yield mean |S| = 1.061 ± 0.659, with only 7% exceeding the classical bound.

**(B)** Cumulative distribution showing optimized configuration at 99th percentile. The large effect size (Cohen's d = 3.68, p < 10⁻²⁰) confirms that high CHSH values are not accidental but require precise parameter tuning.

Dashed line: classical bound |S| = 2; dotted line: Tsirelson bound 2√2 ≈ 2.828.

**Location**: `/paper/figures/figS1_control_random.png`

**Experiment**: C1 (Control: Random Parameter Sweep)

**Key message**: High CHSH values require deliberate parameter optimization - random parameter choices overwhelmingly yield classical correlations.
