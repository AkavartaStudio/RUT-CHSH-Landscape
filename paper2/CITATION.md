# How to Cite This Work

This repository accompanies the manuscript:

**"Echo Geometry of the Classical CHSH Ridge:
Memory, Curvature, and Directional Susceptibility (Paper 2: The Memory Landscape)"**
**Author:** Kelly McRae
**Status:** Preprint (December 2025)

## Manuscript Citation (Preprint)

If you reference the scientific results, please cite the preprint:

**BibTeX**
```bibtex
@misc{McRae2025MemoryLandscape,
  title={Echo Geometry of the Classical {CHSH} Ridge:
         Memory, Curvature, and Directional Susceptibility (Paper 2: The Memory Landscape)},
  author={McRae, Kelly},
  year={2025},
  month={12},
  note={Preprint},
  howpublished={\url{https://github.com/AkavartaStudio/RUT-CHSH-Landscape}},
}
```

**Text citation:**
McRae, K. (2025). *Echo Geometry of the Classical CHSH Ridge:
Memory, Curvature, and Directional Susceptibility (Paper 2: The Memory Landscape).* Preprint.

## Citing Specific Results

If you want to cite specific findings from the study, please reference them within context, e.g.:

* **Universal memory threshold:** σ_mem ≈ 0.002
* **Fragility ratio:** σ_mem/σ_c ≈ 0.03–0.08 (~30× below CHSH collapse)
* **Memory curvature:** C_mem range [-0.005, +0.007]
* **Directional susceptibility:** χ_mid ~85% positive across (K, σ) plane

All results are directly reproducible using the scripts in `experiments/Paper2_Stage*/`.
