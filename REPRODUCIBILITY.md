# Reproducibility Guide

This document provides instructions for reproducing all results presented in:

**"CHSH Correlation Landscape in Coupled Noisy Oscillators"**
*Paper 1: Parameter-Space Mapping and Temporal Memory*

## System Requirements

- **Python:** 3.8 or higher
- **Required packages:** `numpy`, `scipy`, `matplotlib`, `json`
- **Compute time:** ~4-6 hours for full reproduction (single core)
- **Storage:** ~500 MB for all output data

## Quick Start

To reproduce all Paper 1 results in one command:

```bash
cd analysis/scripts/paper1_runners
bash RUN_ALL_PAPER1.sh
```

This will execute all four experiments (A1, A2, A3, B1) sequentially and generate:
- Raw data files in `analysis/data/paper1/`
- Figure outputs in `paper/figures/`

```bash
cd paper/figures/scripts

```

## Core Simulation Engine

The underlying dynamics are implemented in:
```
analysis/scripts/rut_core.py
```

Key functions:
- `integrate_rut()`: Solves coupled oscillator equations with noise
- `compute_chsh()`: Evaluates continuous CHSH functional
- `compute_phase_coherence()`: Calculates phase-locking order parameter r
- `compute_autocorrelation()`: Measures temporal coherence ρ_S(τ)

## Expected Results

Running the full pipeline should produce results matching the paper within statistical uncertainty:

| Quantity | Paper Value | Expected Range |
|----------|-------------|----------------|
| Max \|S\| | 2.819 ± 0.003 | 2.81 - 2.83 |
| Optimal K | 0.7 | (fixed) |
| Optimal σ | 0.2 | (fixed) |
| Optimal Δω | 0.2 | (fixed) |
| Collapse slope | 0.60 ± 0.02 | 0.58 - 0.62 |
| Collapse intercept | 0.22 ± 0.01 | 0.21 - 0.23 |
| Fit R² | > 0.98 | > 0.97 |

Variations within ±5% are typical due to stochastic noise realizations across different random seeds.

## Troubleshooting

**Issue:** Import errors for `rut_core`
**Solution:** Ensure `analysis/scripts/` is in your Python path or run from repo root

**Issue:** Figures don't match paper exactly
**Solution:** Random seed variations are expected; statistical properties should match within error bars

**Issue:** Runtime exceeds estimates
**Solution:** Runtime scales with CPU performance; estimates assume modern multi-core processor

**Issue:** Missing matplotlib backend
**Solution:** Install with `pip install matplotlib` or use system package manager
