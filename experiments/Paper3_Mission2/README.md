Identify multi-basin behavior, history-dependent states, and sensitivity
to phase initialization in the three-oscillator triangle.

- **E324–E327**: Biased Loop-Charge Initializations
- **E328–E330**: Braid-Bait Starts

## Outputs

- Initialization → attractor mapping
- Cross-run repeatability indicators
