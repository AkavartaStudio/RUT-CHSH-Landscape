# Paper 3 - Mission 4: Path-Order & Contextuality Tests (E381–E399)

## Purpose
Test classical contextuality-like behavior by showing that the order of
measurement (A→B→C vs A→C→B vs C-first) changes observed S-like metrics.

## Experiments
- **E381**: A→B→C Sampling
- **E382**: A→C→B Sampling
- **E383**: C-First Contextual Shifts
- **E384–E390**: Multi-Angle Order Tests (6–9 angles)
- **E391–E395**: Two-Body Reduction Comparisons
- **E396–E399**: Classical Contextuality Benchmarks

## Outputs
- S-metric under all sampling paths
- ΔS(order) contextuality curves
- Full 3-body vs 2-body reduction comparisons
